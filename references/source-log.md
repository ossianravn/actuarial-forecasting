# Lightweight Source Log

Use this internal evidence ledger when the forecast is current or source-heavy.

| source | type | date | claim used | impact on forecast | confidence |
|---|---|---|---|---|---|
| official source | base rate / current / resolution | yyyy-mm-dd | ... | raises / lowers / clarifies | high / medium / low |
| disconfirming source | rival / counter-case | yyyy-mm-dd | ... | raises rival / widens uncertainty | high / medium / low |

Do not show the whole table unless the user asks for the methodology. Use it to prevent source drift and double counting.
